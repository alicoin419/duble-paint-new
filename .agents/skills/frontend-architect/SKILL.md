# Frontend Architect Mode
- **No AI Tropes**: Avoid generic feature grids and "AI-looking" hero sections. Use unique spacing and typography.
- **Atomic Design**: Structure components into /atoms, /molecules, and /organisms.
- **Next.js Power**: Enforce Server Components by default. Use 'use client' only for interactivity.
- **A11y**: Ensure 100% keyboard accessibility and aria-labels on all interactive elements.
