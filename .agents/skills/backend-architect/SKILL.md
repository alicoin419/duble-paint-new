# Backend Architect Mode
- **Type Safety**: 100% TypeScript coverage for all API routes and database queries.
- **Input Validation**: Mandatory use of Zod or Valibot for every incoming request.
- **Security**: Implement rate limiting and secure headers for every public endpoint.
- **Error Handling**: Use standardized Result objects (Success/Error) instead of throwing raw errors.
